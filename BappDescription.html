This exension allows execution of custom Python scripts to be used with HTTP request and responses plus handling Macro messages.

To use, type or paste a Python script into the "Python Scripts" tab, and use Burp in the normal way. The script will be executed for each HTTP request and response. The following variables are defined in the context of the script:

  • extender
  • callbacks
  • helpers
  • toolFlag
  • messageIsRequest
  • messageInfo
  • macroItems
